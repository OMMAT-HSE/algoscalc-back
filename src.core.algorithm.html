src.core.algorithm module

class src.core.algorithm.Algorithm(name: str, title: str, description: str, log_config: dict[str, Any], execute_timeout: int = 5)

Базовые классы: object

Класс представляет описание алгоритма, структуры входных и выходных данных, предоставляет возможность выполнения алгоритма согласно заданным параметрам.

add_execute_method(method: Callable) → None

Добавляет метод, реализующий выполнение алгоритма.

Параметры:

method (Callable) – метод, реализующий выполнение алгоритма;

Результат:

None

Исключение:

  • TypeError – если параметр не является методом;

  • RuntimeError – если тестовое выполнение алгоритма завершилось с ошибкой.

add_output(output: DataElement) → None

Добавляет в описание выходных данных алгоритма, элемент переданный в параметре output

Параметры:

output (DataElement) – элемент выходных данных.

Результат:

  • ValueError – если элемент выходных данных с указанным именем уже имеется в описании структуры выходных данных.

execute(params: dict[str, Any]) → dict[str, Any]

Выполняет алгоритм с заданными входными данными.

Параметры:

params (dict[str, Any]) – значения входных данных для выполнения алгоритма. Словарь, где ключи - имена входных данных, значения - фактические значения входных данных.

Результат:

результат выполнения алгоритма. Словарь, где ключи - имена выходных данных, значения - рассчитанные значения выходных данных.

Тип результата:

dict[str, Any]

Исключение:

  • TypeError – если не установлен метод для выполнения, если параметры переданы не словарем, если метод вернул результаты не в виде словаря;

  • KeyError – если во входных или выходных данных отсутствует необходимый или имеется лишний элемент;

  • TimeoutError – если закончилось время, отведенное для выполнения алгоритма;

  • RuntimeError – при возникновении ошибки при выполнении.
